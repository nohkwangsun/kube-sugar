KUBE-SUGAR

---

kube-sugar is the set of commands that allow you to send kubectl commands more easily.

---

```
$ kubectl get pod --namespace default

$ kubectl get svc --namespace default

...
```

kube-sugar can shorten the kubectl commands.

```
$ ks.get.pod

$ ks.get.svc

...
```

kube-sugar

 1. ns.get
 2. ns.set
 3. mk.get
 4. mk.set
 5. ks.get.()()()
 6. ks.exe.()()()
 7. ks.all.()()()
